Files: 
               ./Demo_Synthetic.R: A demo for synthetic datasets. 
               ./Demo_HMP.R: A demo for HMP datasets.
               ./MPLasso: R package for MPLasso
-------------------------------------------------------------------------------

Installation:
Method 1: direct download:
Step 1. Download MPLasso source code and use R to open ./MPLasso/MPLasso.Rproj 
Step 2. Build and Reload MPLasso
Method 2: use devtools:
devtools::install_github('ChiehLo/MPLasso_RPackage/MPLasso')

Demo scripts:
source Demo_Synthetic.R or Demo_HMP.R in Rstudio.
